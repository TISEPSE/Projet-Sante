# Projet Santé

Ce projet est une application de gestion de santé composée d'un backend Flask et d'un frontend moderne.

## Structure du Projet

- **/backend** : API REST développée avec Flask, SQLAlchemy (SQLite) et JWT pour l'authentification.
- **/frontend** : Interface utilisateur (React/Vite).
- **/Doc du projet** : Documentation et annexes du projet.

## Installation Rapide

### Backend
1. Allez dans le dossier `backend`.
2. Installez les dépendances : `pip install -r requirements.txt`.
3. Initialisez la base de données : `python init_db.py`.
4. Lancez le serveur : `python app.py`.

### Frontend
1. Allez dans le dossier `frontend`.
2. Installez les dépendances : `npm install`.
3. Lancez le projet : `npm run dev`.
